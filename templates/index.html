This is a system that allows you and your friends to check-in to an event together. Then if you test positive for COVID-19, let us know and we'll tell everyone else anonymously.

üéâ How do I use this as an attendee?

  1. Scan the QR Code and send the SMS to the number. We'll send you a message confirming you checked in. Show this to the host to get in.
  2. If you test positive, simply reply POSITIVE. We'll tell other attendees without mentioning you in any way.
  3. If someone in an event you attended tested positive, we'll send you an SMS without any identifying info about the positive case.

üìã How do I use this page as an event host?

  1. Print this page or take a screenshot of it.
  2. Have each attendee scan the QR Code and send the SMS when entering your event.

Other Questions

How is close-contact defined?

We will notify you if someone who told us they test positive scanned the same QR code within three hours of you. This is imperfect.

How is my data stored?

  • I use Google App Engine, and store the ID that identifies each event alongside the phone number.
  • I'm going to implement a job to scrub all data after 14 days.

Why does this exist?

Getting COVID sucks. Even worse still, it's really hard to phone up or contact people you care about telling them that you've gotten it. Even though there shouldn't be, there is a certain stigma attached to getting COVID. We live in a pandemic, but somehow when we get sick as individuals, it must be because we did something wrong individually that wrought this disease upon us. This is bad both because it is crippling to our emotional wellbeing, but also because it is terrible for public health.

By making it so that the process for notifying people you tested positive is more frictionless and anonymous, we hope to make it less anxiety-inducing to report a positive case. You have enough to worry about if you test positive.

What was this inspired by?

In large part, this website is a simple, hacky clone of the National SMS Contact Tracing System currently used in Taiwan, except on a much smaller scale, between friends.
